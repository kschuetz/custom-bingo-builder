How Templates Work

  • The template engine will find the first <div> inside the body that has the class card-template. Everything inside this <div> represents a single card and will be repeated on the output.
  • Inside the template, any html elements with the class insert-word will have its contents replaced with a random item from the supplied word list. For example: <span class="insert-word"></span>

Limitations

  • Templates must begin with <!DOCTYPE html>
  • Maximum template size is @settings.maxTemplateSizeBytes bytes.
  • Most common HTML elements are permitted, but certain types will be removed. Hyperlinks, <script> tags, and <link> tags will be removed.
  • All JavaScript will be removed.
  • <style> tags and inline styles are permitted.
  • <img> tags are permitted, but the src attribute must point to an absolute URL that begins with // . Images with relative URLs will be removed.
